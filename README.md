
# GPT Chat Telegram Bot

Это Telegram-бот, который использует GPT для общения. Бот построен с использованием TypeScript и следует модульной архитектуре с применением принципов Feature-Sliced Design (FSD).

## Возможности

- Команды для старта и получения справки
- Команда эхо, которая повторяет ввод пользователя
- Динамическое управление командами с опциями меню
- Модульная структура с использованием принципов FSD
- Поддержка TypeScript для безопасности типов

## Установка

1. Клонируйте репозиторий:

   ```bash
   git clone https://github.com/PanDmitriy/gpt-chat-tg-bot.git
   ```

2. Перейдите в каталог проекта:

   ```bash
   cd gpt-chat-tg-bot
   ```

3. Установите зависимости:

   ```bash
   npm install
   ```

## Использование

### Разработка

Чтобы запустить бота в режиме разработки с горячей перезагрузкой:

```bash
npm run dev
```

### Продакшен

Чтобы скомпилировать и запустить бота в продакшен-режиме:

```bash
npm run build
npm start
```

## Команды

- `/start`: Запустить бота и получить приветственное сообщение.
- `/help`: Получить список доступных команд.
- `/echo <сообщение>`: Повторить сообщение, введенное пользователем.
- `/menu`: Показать главное меню с опциями.

## Структура каталога

```
src/
├── app/
│   └── index.ts          # Точка входа в приложение
├── entities/
│   └── user/             # Сущность пользователя и сервисы
├── features/
│   ├── echo/             # Функционал команды эхо
│   ├── help/             # Функционал команды справки
│   └── start/            # Функционал команды старта
├── shared/               # Общие утилиты и типы
└── widgets/              # UI-компоненты и интерактивные функции
```

Создайте файл `.env` в корневом каталоге для настройки переменных окружения. Пример:

```
BOT_TOKEN=ваш_токен_телеграм_бота
```
